To let the Amazon rainforest system run, you need to follow these steps:

1) Copy the line from the file "r_crit_unstable_amazon_submit.txt" into the shell to start the main file (This is test data, the real data can be requested by the corresponding author from the paper "Forest-rainfall cascades buffer against drought across the Amazon" (Nature Climate Change, 2018))
2) The results will be saved to results folder and can be evluated with the script (r_crit_risk_maps.py) in the evaluations folder
